# Política de Privacidad – SilentBots Studio
Última actualización: 2025-01-01

Esta Política de Privacidad explica qué datos recopilan nuestros bots y cómo los utilizamos. Nos comprometemos a manejar la información de manera responsable y respetando las reglas de Discord.

## 1. Información que Recopilamos
Nuestros bots pueden recopilar y almacenar únicamente la información necesaria para funcionar:

### Datos básicos:
- ID del servidor
- ID de usuarios
- ID de canales y roles
- Configuraciones hechas por administradores
- Parámetros relacionados con comandos o módulos del bot

### Datos opcionales (según permisos):
- Presencia del usuario (online/offline)
- Información de miembros (si el bot usa Members Intent)
- Contenido de ciertos mensajes solo si la función lo requiere (ej. filtros, reportes o auditorías configuradas)

## 2. Datos que NO Recopilamos
Los bots NO almacenan:
- Conversaciones privadas completas
- Contraseñas
- Correos electrónicos
- IPs
- Información personal sensible
- Datos no proporcionados por la API de Discord

Nunca vendemos ni compartimos información con terceros.

## 3. Cómo Usamos tu Información
La información recopilada se usa para:
- Guardar configuraciones del servidor
- Ejecutar correctamente las funciones del bot
- Prevenir abusos o ataques
- Mejorar estabilidad y rendimiento
- Resolver errores técnicos

## 4. Eliminación de Datos
Puedes solicitar la eliminación de todos los datos asociados a tu servidor o usuario.

Métodos:
1. Contactarnos por Discord
2. Retirar el bot del servidor (en algunos bots la eliminación es automática)

## 5. Seguridad de la Información
Implementamos medidas de protección como:
- Bases de datos protegidas
- Acceso limitado solo al desarrollador
- Encriptación cuando es necesario
- Monitoreo para evitar abuso

Aunque aplicamos buenas prácticas, no podemos garantizar seguridad absoluta.

## 6. Servicios de Terceros
Los bots pueden conectarse con:
- API oficial de Discord
- API de Albion Online (solo en bots relacionados)

Ninguno de estos servicios solicita información personal privada.

## 7. Cambios en esta Política
Podemos actualizar o modificar esta política en cualquier momento. Las modificaciones serán efectivas desde su publicación en esta página.
