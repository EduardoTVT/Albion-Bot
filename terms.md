# Términos de Servicio – SilentBots Studio
Última actualización: 2025-01-01

Bienvenido/a. Estos Términos de Servicio regulan el uso de los bots desarrollados por SilentBots Studio, incluyendo (pero no limitado a): AlbionBOT

Al añadir o utilizar cualquiera de nuestras aplicaciones en tu servidor de Discord, aceptas completamente estos términos. Si no estás de acuerdo, deberás retirar la aplicación del servidor.

## 1. Aceptación de los Términos
Al usar nuestros bots, confirmas que:
- Cumples las políticas de Discord
- Eres mayor de la edad requerida por Discord en tu región
- Tienes permiso para gestionar el servidor donde se usa la aplicación

## 2. Uso del Servicio
El usuario se compromete a NO utilizar los bots para:
- Actividades maliciosas o ilegales
- Spam, acoso o abuso
- Eludir restricciones de Discord
- Automatizaciones prohibidas o no permitidas en juegos externos

Puedes utilizar las funciones siempre que respetes las reglas del servidor y la API de Discord.

## 3. Permisos dentro del Servidor
Al invitar los bots, otorgas permisos como:
- Leer y enviar mensajes
- Gestionar roles o canales (solo si tú lo habilitas)
- Ver IDs de usuarios y configuraciones del servidor
- Usar Privileged Intents si están habilitados (presencia, contenido de mensajes, miembros)

Los bots solo acceden a datos que Discord permite mediante su API oficial.

## 4. Recolección y Almacenamiento de Datos
Nuestros bots pueden almacenar:
- IDs de servidor
- IDs de usuario
- Roles, canales y configuraciones
- Comandos ejecutados
- Logs básicos de funcionamiento y seguridad

No se almacena contenido privado, como:
- Mensajes completos (a menos que una función explícita lo requiera)
- Contraseñas
- Correos electrónicos
- Información sensible

## 5. Disponibilidad del Servicio
SilentBots Studio puede:
- Actualizar funciones sin previo aviso
- Suspender el bot temporalmente por mantenimiento
- Modificar o eliminar características para mejorar estabilidad
- Presentar caídas imprevistas

No garantizamos disponibilidad 24/7, aunque trabajamos para ello.

## 6. Limitación de Responsabilidad
No nos hacemos responsables de:
- Daños ocasionados por mal uso del bot
- Problemas en tu servidor derivados de configuraciones incorrectas
- Eliminación de datos accidental por parte del administrador del servidor
- Acciones tomadas por Discord hacia tu servidor o cuenta

Usas el bot bajo tu propio riesgo.

## 7. Terminación del Servicio
Podemos suspender o restringir el uso del bot si:
- Se detecta abuso
- Se incumplen estos términos
- Discord solicita acción por violación de sus políticas
